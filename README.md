# EchoJournal

# In a Nutshell
EchoJournal is a colorful audio journaling app. It ºs aim is to help users to quickly log their thoughts and mood throughout the day by recording voice memos.

# Tech Stack
- [x] Jetpack Compose
- [X] Compose KMP (Android, iOS) 
- [x] Koin dependency injection
- [x] Clean architecture
- [x] Gradle version catalog
- [x] kotlin coroutines
- [x] Flows
- [x] MVVM
- [X] Room database
- [x] Material design 3
- [X] Recording audio https://github.com/theolm/kmp-record
- [X] Audio playback https://github.com/LexiLabs-App/basic/tree/main/basic-sound
